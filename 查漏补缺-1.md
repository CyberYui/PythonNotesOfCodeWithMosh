## 2. 数据结构操作

### 2.1 列表操作
- **问题：** insert()方法使用错误
- **正确用法：**

### 2.2 集合和字典
- **问题：** 用()创建集合，混淆zip()用法
- **正确创建：**

### 2.3 列表推导式
- **问题：** 不会使用列表推导式过滤偶数
- **正确做法：**
```python
# 错误做法
def get_evens(numbers):
    result = []
    for num in numbers:
        if num % 2 == 0:
            result.append(num)
    return result

# 正确做法（列表推导式）
def get_evens(numbers):
    return [x for x in numbers if x % 2 == 0]
```

### 4.3 属性装饰器
- **掌握情况：** ✅ 良好
- **完整示例：**
```python

    def __init__(self, name):
        self._name = name
    
    @property
    def name(self):
        return self._name

    def name(self, value):
        if isinstance(value, str):
            self._name = value
        else:
            raise ValueError("Name must be a string")
```

## 6. 高级特性

### 6.1 map和filter
- **问题：** 混淆map操作
- **正确用法：**
```python

numbers = [1, 2, 3, 4, 5]
```

### 6.3 上下文管理器
- **问题：** 不会实现自定义上下文管理器
- **实现方法：**
```python
class FileManager:
    def __init__(self, filename, mode):
        self.filename = filename
        self.mode = mode
    
    def __enter__(self):
        self.file = open(self.filename, self.mode)
        return self.file
    
    def __exit__(self, exc_type, exc_val, exc_tb):
        self.file.close()

# 使用
with FileManager('test.txt', 'r') as f:
    content = f.read()
```

## 7. 作用域和全局变量

### 7.1 global关键字
- **问题：** 错误理解global的用途
- **正确用法：**
```python
# global用于在函数内修改全局变量
count = 0

def increment():
    global count  # 声明使用全局变量
    count += 1

# 错误用法：global不能用在if、for等块中

    global x  # SyntaxError!
```

### 7.2 nonlocal关键字
- **用于嵌套函数中**
```python
def outer():
    x = 10
    def inner():
        nonlocal x  # 修改外层函数的变量
        x += 1
        return x
    return inner
```

## 11. 多态

### 11.1 什么是多态
- **定义：** 同一接口，不同实现
- **核心思想：** 不同对象对同一消息有不同的响应

### 11.2 实现多态的方式
1. **继承重写**
```python
class Animal:
    def speak(self):
        raise NotImplementedError

class Dog(Animal):
    def speak(self):
        return "Woof!"

class Cat(Animal):
    def speak(self):
        return "Meow!"

# 多态调用
def make_animal_speak(animal):
    print(animal.speak())

make_animal_speak(Dog())  # Woof!
make_animal_speak(Cat())  # Meow!
```

2. **鸭子类型**
```python
class Duck:
    def quack(self):
        return "Quack!"

class Person:
    def quack(self):
        return "I'm quacking like a duck!"

# 鸭子类型：如果有quack方法，就是鸭子
def in_the_forest(duck):
    print(duck.quack())

in_the_forest(Duck())      # Quack!
in_the_forest(Person())    # I'm quacking like a duck!
```

### 11.3 多态的优势
- 代码灵活性高
- 易于扩展
- 符合开闭原则

## 12. 上下文管理器

### 12.1 上下文管理器协议
- **必须实现的方法：**
  - `__enter__()`
  - `__exit__()`

### 12.2 手动实现上下文管理器
```python
class DatabaseConnection:
    def __init__(self, db_url):
        self.db_url = db_url
        self.connection = None
    
    def __enter__(self):
        print("Connecting to database...")
        self.connection = f"Connected to {self.db_url}"
        return self.connection
    
    def __exit__(self, exc_type, exc_val, exc_tb):
        print("Closing database connection...")
        self.connection = None
        # 如果有异常，可以在这里处理
        if exc_type is not None:
            print(f"Exception occurred: {exc_val}")
        return True  # 抑制异常

# 使用
with DatabaseConnection("localhost:5432") as conn:
    print(f"Using connection: {conn}")
```

### 12.3 使用contextlib装饰器
```python
from contextlib import contextmanager

@contextmanager
def file_manager(filename, mode):
    print(f"Opening file {filename}")
    file = open(filename, mode)
    try:
        yield file
    finally:
        print(f"Closing file {filename}")
        file.close()

# 使用
with file_manager("test.txt", "w") as f:
    f.write("Hello, World!")
```

## 13. isinstance() 和 type() 的区别

### 13.1 基本区别
- **isinstance()**: 考虑继承关系
- **type()**: 不考虑继承，只检查确切类型

### 13.2 使用示例
```python
class Animal: pass
class Dog(Animal): pass

dog = Dog()

# isinstance() - 考虑继承
print(isinstance(dog, Dog))      # True
print(isinstance(dog, Animal))   # True
print(isinstance(dog, object))   # True (所有类都继承自object)

# type() - 不考虑继承
print(type(dog) == Dog)         # True
print(type(dog) == Animal)      # False
print(type(dog) == object)      # False
```

### 13.3 使用场景
- **使用 isinstance()**:
  - 类型检查时需要考虑继承
  - 检查多个类型：isinstance(x, (int, str))
  - 检查抽象基类

- **使用 type()**:
  - 需要精确类型匹配
  - 不想考虑继承关系
  - 调试时查看确切类型

### 13.4 最佳实践
```python
# 推荐：使用 isinstance 进行类型检查
def process_value(value):
    if isinstance(value, (int, float)):
        return value * 2
    elif isinstance(value, str):
        return value.upper()
    else:
        raise TypeError("Unsupported type")

# 不推荐：使用 type 进行类型检查
def process_value_bad(value):
    if type(value) == int or type(value) == float:
        return value * 2
    # ... 其他判断
```
