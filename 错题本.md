### Question 4
**题目内容：** Write a conditional statement that checks if a number `num` is positive and prints "Positive" if it is.

**我的答案：** 
```python
if(num>=0):
    print("Positive")
```
**正确答案：** 
```python
if num > 0:  # 注意：正数应该是 > 0，而不是 >= 0
    print("Positive")
```
**错误原因：** 0 不是正数，应该是严格大于0

### Question 7
**题目内容：** Write a for loop that prints numbers from 1 to 5.

**我的答案：** 
```python
for i in range(1,5)
    print(i)
```
**正确答案：** 
```python
for i in range(1, 6):  # range是左闭右开的，5需要写成6
    print(i)
```
**错误原因：** range函数的上限不包含，应该使用6而不是5

### Question 3
**题目内容：** Write a function that takes a list of numbers and returns only the even numbers using list comprehension.

**我的答案：** 
```python
def even_in_list(list):
    even = []
    for item in list:
        if item//2:
            even.insert(item)
        else:
            continue
    return even
```
**错误原因：**
1. list是Python关键字，不应作为变量名
2. 偶数判断错误：item//2!=0不表示偶数，应该用x % 2 == 0
3. insert用法错误：insert(index, value)，不是insert(value)
4. 没有用到列表 comprehension

### Question 8
**题目内容：** Write a class `Person` with attributes `name` and `age`, and a method `greet()` that returns "Hello, my name is {name}".

**我的答案：** 
```python
class Person(self):
    self.name = ""
    self.age = 0
    
    def greet(name):
        return f"Hello, my name is {name}"
```
**错误原因：**
1. 类定义不需要self参数
2. 属性应该在__init__方法中初始化
3. greet方法应该有self参数
4. 使用self.name访问实例属性

### Question 10
**题目内容：** Write a try-except block to handle division by zero error.

**我的答案：** 
```python
number = 4
try:
    number/0
Except ZEROERROR as ze:
    print("Wrong statement ! 0 can't be the division.")
```
**错误原因：**
1. 异常类名应该是ZeroDivisionError，不是ZEROERROR
2. Python关键字大小写敏感：except而不是Except
3. 异常处理应该用冒号而不是as ze（除非需要捕获异常对象）
